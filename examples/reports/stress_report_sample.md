## 核心结论

1. 高频轻量场景首选「mini」类模型：RPS 3+、100% 成功率、成本可控。
2. 极致性价比选 Flash-Lite 类模型：每美元 340 万 tokens 的量级。
3. 高质量任务选 Sonnet / Opus：零失败，但延迟与成本较高。
4. 推理任务优先选「fast-reasoning」：推理模型中的吞吐王。
5. 生产环境需慎用仅支持 Responses API 且超时率高的模型。
